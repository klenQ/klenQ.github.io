    笔记本安装Linux+Windows10双系统

    Linux 使用 Ubuntu 20.10LTS

    使用环境:

    DELL G3 3500 笔记本

    显卡为 Nvidia 1650ti

    CPU Intel(R) Core(TM) i5-10300H CPU @ 2.50GHz

    内存 8G

    Windows 10 HOME x64

    前置准备

    主要分为一下几步

    • 下载镜像

    • 制作系统安装盘

    • 电脑分区

    • 插入安装盘并设置Bios并重启

    下载镜像

    从ubuntu官网下载对应系统镜像, .iso文件

    ubuntu官网

    制作系统安装盘

    最好准备8G或以上的空U盘, 因为制作安装盘里面内容会被抹除

    这一步很简单, 具体搜索一下就好了, 可以使用类似UltraISO软通碟等U盘刻录软件将.iso文件作为镜像刻录到U盘

    电脑分区

    使用DiskGenius的软件为电脑腾出需要分配给Linux系统的分区, 如果不用动C盘的容量最好, 这里因为只有一个硬盘所以使用了C盘

    具体操作可以自行搜索一下

    需要注意的是, 如果硬盘使用了BitLocker需要先在设置中解除才能用第三方软件进行分区.

    安装

    如果出现卡死的情况, 参照这篇博客笔记本装双系统！win10+Linux！所有的坑自己一个个爬过来，纪念一下。

    首先需要关闭windows的RST功能(Rapid Storage), 因为Linux不支持

    Windows doesn’r boot after disabling Intel Rapid Storage

    关于如何进入安全模式Safe Mode How to Boot to Safe Mode in Windows 10 and Windows 11

    如果出现perform MOK management 的界面,

    [安装Ubuntu后重启出现perform MOK management]

    启动安装盘

    将U盘插入电脑, 然后重启进入BIOS界面,

    DELL一般是使用F2进入BIOS界面

    如果是旧版BIOS设置启动设备就好了, 新版BIOS需要选择到具体启动的.efi文件.

    我们的启动文件在安装盘(一般路径名字叫Ubuntu)下的/EFI/BOOT/BOOTx64.EFI

    将启动设备/路径选择好了之后, 更改顺序, 将安装盘的启动设置移到第一位

    这时候退出BIOS, 电脑就会启动, 并进入安装界面

    Linux系统安装

    选择语言, 选择输入法, 直到出现选项选择安装方式

    可以选择install alongside with Windows就是自动安装到之前分配出来的空分区

    也可以使用手动安装

    等待安装完毕, 安装完毕后重启

    正常重启可能依旧会进入到windows系统

    首先确认关闭Window电源选项中的快速启动
