    真实DOM和虚拟DOM

    虚拟DOM:

    1. 本质是Object类型对象
    2. 虚拟DOM比较轻
    3. 最终会被转化为真实DOM

    JSX

    1. 全程 JavaScript XML
    2. 用来简化创建虚拟DOM
      1. 最终产生的就是一个js对象
      2. 不是字符串也不是HTML/XML标签
    3. 标签名任意, HTML标签或者自定义标签
    4. 标签中用JS表达式的时候需要用{}
    5. class指定要用className
    6. style内联样式需要用stlye={{key:value}}的方式写
    7. 只能有一个根标签, 标签必须闭合
