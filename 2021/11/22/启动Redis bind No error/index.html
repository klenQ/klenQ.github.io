启动Redis redis-server.exe无法启动或者报错
2021-11-22|Solutions

启动Redis文件夹中 redis-server.exe 控制台一闪而过

运行环境:

Redis Windows 64位版

Redis-x64-3.2.100

Redis官网

Window下启动redis服务报错

报错 Creating Derver TCP listening socket 127.0.0.1:6379: bind No error

解决方案:

到redis根目录下进入cmd控制台

输以下指令关闭redis-cli.exe

  1. redis-cli.exe
  2. shutdown
  3. exit
  4. redis-server.exe redis windows.conf
